# UniGen sampling demo

This experiment connects the WFC-to-SAT compiler output to UniGen 2.7.0.

Results so far:

- UniGen accepted existing DIMACS CNFs.
- Five Simple Wall samples were generated and decoded.
- Seven Chess samples were generated and decoded.
- Outputs show strong diversity but lower source-image fidelity than a weighted sampler would ideally preserve.

The UniGen binary itself is not included. It was downloaded from the official Meel Group release.
